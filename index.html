Quick start is quick:

~ $ gem install jekyll

~ $ git clone git@github.com:lenciel/django-best-practices.git

~ $ cd django-best-practices/site

~/site $ jekyll serve --watch

# => 打开浏览器浏览 http://localhost:4000

Jekyll站点可以直接部署到GitHub Pages

使用GitHub Pages 可以host任何使用 Jekyll编写的站点，你可以部署站点到Github甚至还可以为这些站点绑定自己的域名。
